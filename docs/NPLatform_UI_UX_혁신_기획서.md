## 2. 모바일 반응형 완전 최적화

### 2.1 글로벌 반응형 시스템
```
파일: styles/globals.css 확장

브레이크포인트 전략:
  Mobile First (기본) → sm (640px) → md (768px) → lg (1024px) → xl (1280px)

글로벌 규칙:
  - 모든 테이블 → 모바일에서 카드 뷰 자동 전환
  - 사이드바 → 모바일에서 Sheet(바텀시트) 변환
  - 멀티컬럼 → 모바일에서 단일 컬럼 스택
  - 터치 타겟 최소 44x44px (WCAG 기준)
  - Safe area inset 적용 (노치/홈바)
```

### 2.2 페이지별 모바일 최적화 상세

#### 검색 페이지 모바일
```
현재: 테이블이 가로스크롤 → 사용 불편
개선:
  - md 이하: 카드 뷰 강제 (테이블 숨김)
  - 카드: 한 줄에 1개, 핵심 정보만 (기관, 주소, 금액, AI등급)
  - 필터: 바텀시트 (Sheet)로 변환 → 전체화면 필터
  - 탭: 가로 스크롤 + 그라데이션 페이드
  - 검색바: 상단 고정 (sticky)
  - 페이지네이션: "더 보기" 버튼으로 변환
```

#### 시뮬레이터 모바일
```
현재: 좌(입력)+우(결과) → 모바일에서 깨짐
개선:
  - 풀스크린 탭 전환: [입력] [결과] [차트] [분석]
  - 입력폼: 아코디언 그룹화 (기본/비용/세금)
  - 차트: 세로 모드 최적화 (가로 스크롤X)
  - 결과 테이블: 카드형 변환
  - 하단 플로팅 버튼: "계산하기"
```

#### 지도 모바일
```
현재: 사이드바+지도 동시 → 겹침
개선:
  - 지도 전체화면 (기본)
  - 하단 반투명 미니카드 (현재 선택 매물)
  - 위로 스와이프 → 매물 리스트 (바텀시트)
  - 지도 터치 제스처: 핀치줌, 더블탭줌
  - FAB 버튼: 내 위치, 필터
```

#### 딜룸 모바일
```
현재: 사이드바+메시지 겹침
개선:
  - 전체화면 채팅 (메신저 스타일)
  - 헤더: 거래명, 상태배지, 뒤로가기
  - 메시지 입력: 키보드 위 고정
  - 파일 첨부: 카메라/갤러리/파일 선택
  - 거래 정보: 상단 탭으로 전환
```

## 3. UI/UX 혁신 기획

### 3.1 다크모드 (Dark Mode)
```
파일:
  tailwind.config.js → darkMode: 'class'
  components/layout/theme-toggle.tsx → 테마 전환 버튼
  app/layout.tsx → ThemeProvider 래핑

색상 매핑:
  Light                    Dark
  bg-white            →   bg-gray-950
  bg-gray-50          →   bg-gray-900
  text-gray-900       →   text-gray-50
  #1B3A5C (primary)   →   #4B8BC9 (밝은 버전)
  #10B981 (accent)    →   #34D399 (밝은 버전)
  border-gray-200     →   border-gray-800

차트:
  Recharts: 다크모드 색상 팔레트
  배경색, 그리드선, 텍스트 모두 전환

저장:
  localStorage 'theme' + 시스템 설정 감지
  prefers-color-scheme 자동 감지
```

### 3.2 마이크로 인터랙션
```
적용 위치:
  - 페이지 전환: Framer Motion layoutId (공유 요소 애니메이션)
  - 카드 호버: scale(1.02) + shadow elevation + 0.2s ease
  - 버튼 클릭: scale(0.97) → scale(1) + ripple effect
  - 토스트: 슬라이드인 + 프로그레스바 + 슬라이드아웃
  - 로딩: 스켈레톤 shimmer (gradient animation)
  - 숫자 변경: 카운팅 애니메이션 (spring physics)
  - 차트 진입: 데이터 포인트 순차 등장
  - 탭 전환: 언더라인 슬라이드 (layoutId)
  - 모달 열기: scale(0.95) → scale(1) + backdrop blur
  - 스크롤: parallax 히어로 (scrollY 기반)

기술:
  Framer Motion: variants, AnimatePresence, layoutId
  CSS: @keyframes, transition, will-change
  Intersection Observer: 스크롤 기반 진입 애니메이션
```

### 3.3 접근성 (Accessibility)
```
WCAG 2.1 AA 기준:
  - 모든 이미지 alt 텍스트
  - 키보드 내비게이션 (Tab, Enter, Escape)
  - ARIA 레이블 (aria-label, aria-describedby)
  - 포커스 표시 (focus-visible:ring)
  - 색상 대비 4.5:1 이상
  - 스크린리더 호환 (role, aria-live)
  - 모션 감소 (prefers-reduced-motion)
  - 폰트 크기 조절 가능 (rem 단위)
```

### 3.4 디자인 시스템 통합
```
파일: lib/design-tokens.ts

토큰:
  colors:
    primary: { 50~950 } // #1B3A5C 기반
    accent: { 50~950 }  // #10B981 기반
    danger: { 50~950 }
    warning: { 50~950 }

  spacing: 4/8/12/16/20/24/32/40/48/64
  radius: sm(4)/md(8)/lg(12)/xl(16)/2xl(24)/full
  shadow: sm/md/lg/xl/2xl

  typography:
    heading: { h1~h6 } (size, weight, lineHeight)
    body: { lg/md/sm/xs }

  animation:
    duration: { fast: 150ms, normal: 300ms, slow: 500ms }
    easing: { ease, spring, bounce }

컴포넌트 표준:
  - 모든 카드: rounded-xl shadow-sm hover:shadow-md transition
  - 모든 버튼: min-h-11 rounded-lg font-medium
  - 모든 입력: h-11 rounded-lg border-gray-200
  - 모든 배지: rounded-full px-2.5 py-0.5 text-xs font-semibold
```

### 3.5 스마트 내비게이션
```
데스크탑:
  - 스크롤 시 네비 축소 (h-16 → h-12, 로고 작아짐)
  - 현재 페이지 하이라이트 (언더라인 애니메이션)
  - 메가 드롭다운: NPL 마켓 hover → 검색/지도/입찰 + 최근 매물 미리보기
  - 검색 바: Cmd+K 글로벌 검색 (매물, 페이지, 기능)

모바일:
  - 하단 탭바: 5탭 (검색/지도/입찰/분석/마이) — 이미 구현
  - 상단: 최소 헤더 (로고 + 알림 + 프로필)
  - 스와이프 제스처: 좌→우 뒤로가기, 우→좌 사이드메뉴
  - 당겨서 새로고침 (pull-to-refresh)

글로벌 검색 (Cmd+K):
  컴포넌트: components/layout/command-palette.tsx
  - Dialog 기반 검색 팔레트
  - 최근 검색, 빠른 이동, 매물 검색
  - 키보드 내비게이션 (↑↓ Enter)
  - shadcn/ui Command 컴포넌트 활용
```

### 3.6 데이터 시각화 혁신
```
차트 업그레이드:
  - 게이지 차트: AI등급/리스크점수 표시 (반원형)
  - 히트맵: 지역별 NPL 밀도 (색상 그라데이션)
  - 퍼널 차트: 전환율 (조회→관심→입찰→매각)
  - 워터폴 차트: 비용 구조 분석
  - 트리맵: 포트폴리오 구성 시각화

인포그래픽:
  - 매물 카드: 미니 스파크라인 (30일 조회수 추이)
  - KPI: 트렌드 화살표 + 변화율 색상
  - 진행률: 원형 프로그레스 (deal stage)
```
