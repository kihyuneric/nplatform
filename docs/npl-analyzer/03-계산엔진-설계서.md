# 03. 계산 엔진 설계서 (lib/calc/)

## 1. 설계 원칙

1. **순수 함수**: 모든 계산 함수는 side effect 없는 순수 함수. 동일 입력 → 동일 출력.
2. **브라우저 실행**: Node.js/서버 의존성 없음. 100% 클라이언트 사이드.
3. **테스트 가능**: 각 함수를 독립적으로 유닛 테스트 가능.
4. **엑셀 수식 1:1 대응**: 기존 엑셀 셀 수식과 완전히 동일한 결과를 보장.

### 4.1 financial-utils.ts

#### `calculateIRR(cashFlows: number[]): number`
- **목적**: 현금흐름 배열에서 IRR(내부수익률) 계산
- **알고리즘**: Newton-Raphson 반복법
- **입력**: `[-초기투자, CF1, CF2, ..., CFn]`
- **출력**: 연간 IRR (소수, 예: 0.183)
- **엣지케이스**:
  - 현금흐름이 모두 양수/음수 → `NaN` 반환
  - 수렴 실패 시 (1000회 초과) → `NaN` 반환
  - 빈 배열 → `0` 반환
- **검증 수식**: `NPV(IRR, cashFlows) ≈ 0` (오차 < 1e-7)

```typescript
export function calculateIRR(cashFlows: number[], guess = 0.1, maxIter = 1000, tolerance = 1e-7): number {
  if (cashFlows.length < 2) return 0;
  let rate = guess;
  for (let i = 0; i < maxIter; i++) {
    let npv = 0;
    let dnpv = 0;
    for (let j = 0; j < cashFlows.length; j++) {
      const denom = Math.pow(1 + rate, j);
      npv += cashFlows[j] / denom;
      if (j > 0) dnpv -= j * cashFlows[j] / (denom * (1 + rate));
    }
    if (Math.abs(npv) < tolerance) return rate;
    if (dnpv === 0) return NaN;
    const newRate = rate - npv / dnpv;
    if (!isFinite(newRate)) return NaN;
    rate = newRate;
  }
  return NaN;
}
```

#### `calculateNPV(cashFlows: number[], discountRate: number): number`
- **수식**: `NPV = Σ(CF[i] / (1+r)^i)` for i=0..n
- 주의: CF[0] (초기투자)도 할인하지 않고 포함 (엑셀 NPV 함수와 다름)

```typescript
export function calculateNPV(cashFlows: number[], discountRate: number): number {
  // 엑셀 NPV 호환: NPV(rate, CF1..CFn) + CF0
  if (cashFlows.length === 0) return 0;
  const cf0 = cashFlows[0];
  const rest = cashFlows.slice(1);
  const npvRest = rest.reduce((sum, cf, i) =>
    sum + cf / Math.pow(1 + discountRate, i + 1), 0);
  return cf0 + npvRest;
}
```

### 4.2 dividend-waterfall.ts

#### `calculateDividendWaterfall(input: DividendInput): DividendResult`

**계산 순서** (엑셀 Sheet② 완전 대응):
```
1. 낙찰가 = input.salePrice
2. 경매비용 = 낙찰가 × auction_cost_rate
3. 당해세 차감
4. 소액임차인 최우선변제 차감
5. 임금채권 차감
6. ★ 배당가용금액 = MAX(0, 낙찰가 - 2 - 3 - 4 - 5)
7. 채권자별 순차 배당 (순위순):
   - 각 채권자 배당 = MIN(채권액, 잔여금)
   - 잔여금 갱신 = MAX(0, 잔여금 - 배당)
8. 임차인 잔여 배당
9. 소유자 잉여금
```

**엑셀 수식 대응표**:
| 엑셀 셀 | 수식 | 함수 내 변수 |
|---------|------|-------------|
| C11 | `=C10*'①기본인풋'!D34` | `auctionCost = salePrice * auctionCostRate` |
| C15 | `=MAX(0,C10-C11-C12-C13-C14)` | `available = Math.max(0, salePrice - costs)` |
| C16 | `=MIN('①기본인풋'!F21, C15)` | `distributed = Math.min(claim.amount, remaining)` |
| C17 | `=MAX(0,C15-C16)` | `remaining = Math.max(0, remaining - distributed)` |

#### `calculateMultiScenarioDividend(scenarios: number[], ...): DividendResult[]`
- 여러 낙찰가에 대해 한번에 배당 계산
- 배열 반환으로 테이블/차트 렌더링에 직접 사용

### 4.3 npl-yield.ts

#### `calculateNplYield(input: NplYieldInput): NplYieldResult`

**수식**:
```
매입가 = 원금 × (1 - 할인율)
순이익 = 배당예상액 - 매입가
절대수익률 = 순이익 / 매입가
연환산IRR = (1 + 절대수익률)^(12/투자개월) - 1
회수율 = 배당예상액 / 원금
MOIC = 배당예상액 / 매입가
```

**엑셀 대응**:
| 엑셀 | 수식 |
|------|------|
| C17 | `=F21*(1-D41)` → purchasePrice |
| C20 | `=E19-C17` → profit |
| C21 | `=IFERROR(E20/C17,0)` → absoluteReturn |
| C22 | `=IFERROR((1+C21)^(12/D38)-1,0)` → annualizedIRR |

#### `calculateNplYieldMatrix(principal, distributions, discountRates, months): NplYieldResult[][]`
- 낙찰가 시나리오 × 할인율 시나리오 전체 매트릭스 계산

### 4.4 auction-irr.ts

#### `calculateAuctionIrr(input: AuctionIrrInput): AuctionIrrResult`

**계산 단계**:

```
[Step 1: 총 취득원가]
  취득세 = 입찰가 × acquisition_tax_rate
  기타비용 = 입찰가 × other_acquisition_rate
  총취득원가 = 입찰가 + 취득세 + 기타 + 명도비 + 보수비

[Step 2: 연간 NOI]
  총임대수입 = 임대단가 × 면적 × 12
  유효임대수입 = 총임대 × (1 - 공실률)
  운영비 = 유효임대 × 운영비율
  NOI = 유효임대 - 운영비

[Step 3: Exit 가치]
  방법A_CapRate = NOI / exit_cap_rate
  방법B_감정상승 = 감정가 × (1 + 상승률)^보유년
  방법C_AI시세 = AI시세 × (1 + 상승률)^보유년    [AI시세 있을 때]
  방법D_배수법 = 감정가 × sale_price_multiplier   [배수 있을 때]
  적용Exit = AVERAGE(방법B, 방법C 또는 방법D)

[Step 4: 현금흐름]
  Year 0: -총취득원가
  Year 1~(N-1): NOI
  Year N: NOI + 적용Exit

[Step 5: IRR/NPV]
  IRR = calculateIRR(cashFlows)
  NPV = calculateNPV(cashFlows, 0.10)
  MOIC = (적용Exit + NOI×보유년) / 총취득원가
```

### 4.5 bulk-npl.ts

#### `calculateBulkNpl(input: BulkNplInput): BulkNplResult`

**계산 단계** (npl_통합모델 Sheet④ 대응):

```
[A. 배당 워터폴]
  경매비용 = 총낙찰가 × 경매비용율
  배당가능액 = 총낙찰가 - 경매비용 - 선순위합계
  예상실배당액 = MIN(배당가능액, NPL최고액합계, NPL채권액_배당일)

[B. 투입자금]
  계약금 = 매입가 × 계약금비율
  잔금 = 매입가 × 잔금비율
  이전비용 = 매입가 × 이전비용율
  비히클사용료 = 매입가 × 비히클사용료율
  중개수수료 = 매입가 × 중개수수료율
  질권대출금 = ROUND(매입가 × LTV, -2)    // 100원 단위 절사
  질권이자 = 질권대출금 × 이자율 × 회수기간일 / 365
  총에쿼티 = 계약금 + 잔금 + 이전비용 + 비히클 + 중개 + 질권이자

[C. 수익성]
  회수금액 = 실배당액 - 질권대출금
  수익금 = 회수금 - 총에쿼티
  기간수익률 = 수익금 / 총에쿼티
  연환산수익률 = 기간수익률 × 365 / 회수기간일

[D. 손실방어]
  손실방어낙찰가 = 총에쿼티 + 선순위합계 + 경매비용
  손실방어낙찰율 = 손실방어낙찰가 / 감정가합계
```

### 4.6 interest-calc.ts

#### `calculateClaimInterest(input: ClaimInterestInput, targetDate: Date): ClaimInterestResult`

**수식** (npl_재정리 Sheet② 대응):
```
1일이자 = 원금 × 연이율 / 365
이자(특정일) = 기산일이자 + 원금 × 연이율 × (특정일 - 기산일) / 365
잔액(특정일) = 원금 + 비용 + 이자(특정일)
배당한도액 = MIN(잔액, 최고액)
```
