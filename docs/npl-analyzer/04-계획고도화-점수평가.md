## 0. 플랫폼 정체성

### NPLatform

### 핵심 가치 흐름

```

 AMC/은행/캠코               NPLatform                   NPL투자자/경매투자자
    │                          │                            │
    ├─ 매물 등록 ──────►  마켓플레이스  ◄────── 매물 탐색 ──┤

    │                          │                            │
    │                    NPL 분석 엔진 ◄──── 수익률 분석 ──┤
    │                   (배당워터폴/IRR/민감도)              │
    │                          │                            │
    ├─ 거래 관리 ──────►    딜룸/계약   ◄────── 투자의향 ──┤
```

## 6. 외부 연동 슬롯 구조

### 연동 방식
```
ExternalSlotConfig {
  type: 'iframe' | 'api' | 'widget'
  url: string        // 관리자가 시스템설정에서 입력
  apiKey: string     // API 인증키
  height: number     // 표시 높이
  enabled: boolean   // 활성화 여부
}
```

### 4개 슬롯
| 슬롯 ID | 유형 | 배치 위치 |
|---------|------|----------|
| `NEWS_FEED` | iframe | 홈 랜딩, 매물 상세 |
| `PSYCHOLOGY_INDEX` | widget | 투자자 대시보드 |
| `MARKET_STATS` | api | 시장 통계 페이지 |
| `KEYWORD_TREND` | iframe | 투자자 대시보드 |

### 설정 방법
관리자 → 시스템 설정 → **외부 연동** 탭에서:
1. URL/API Endpoint 입력
2. API Key 입력 (필요 시)
3. 활성화 Switch ON
4. "연동 테스트" 버튼으로 검증
