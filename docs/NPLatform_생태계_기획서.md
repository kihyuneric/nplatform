# NPLatform 생태계 기획서
## "혼자 만드는 플랫폼이 아닌, 모든 이해관계자가 함께 움직이는 생태계"

---

## 핵심 철학

> 플랫폼 혼자 성장하는 것이 아니라,
> **모든 참여자의 노력이 플랫폼의 성과로 연결**되고,
> **플랫폼의 성과가 다시 모든 참여자에게 돌아가는** 선순환 구조

---

## 3. 역할별 전용 기능

### 3.1 금융기관/AMC 전용
```
/seller/onboarding      — 기관 온보딩 (유형: 은행/저축은행/보험/캐피탈/AMC)
/seller/dashboard       — 매각 현황 대시보드
/seller/listings/new    — 6단계 매물 등록
/seller/portfolio/new   — 포트폴리오 번들 (위탁 물량 일괄)
/seller/analytics       — 매물 성과 분석
/seller/settlement      — 수수료 정산
/seller/bidding         — 받은 입찰 관리, 낙찰 확정
/seller/marketing       — 추천매물 신청, 태그, 공유
```

### 3.2 대부업체 전용 (매도+매수 양면)
```
매도 기능: /seller/* (금융기관과 동일)
매수 기능: /buyer/* (투자자와 동일)
특화:
  /lender/dashboard     — 양면 대시보드 (매도+매수 통합)
  /lender/portfolio     — 보유 NPL 포트폴리오 관리
  /lender/exchange      — 거래소에서 매도/매수
```

### 3.3 자산운용사 전용
```
/fund/dashboard         — 펀드 대시보드 (AUM, 수익률, 포트폴리오)
/fund/portfolio         — 펀드 포트폴리오 관리
/fund/analysis          — 대형 포트폴리오 일괄 분석
/fund/report            — LP(투자자) 보고서 자동 생성
/fund/compliance        — 규정 준수 체크리스트
```

### 3.4 경매/공매 투자자 전용

```
/auction/calendar       — 매각기일 캘린더
/auction/history        — 내 입찰/낙찰 이력
/auction/analysis       — AI 낙찰가율 예측
```

### 3.5 법무법인 전용

```
/professional/law/services    — 등록한 서비스 관리
  - 권리분석 서비스 (건당 50~200만원)

  - 명도소송 대행
  - 배당금 청구 대행

서비스 연결:
```

## 4. 전문 서비스 마켓플레이스

### 4.1 서비스 마켓 (/marketplace/services)
```
모든 전문 서비스를 한 곳에서:

카테고리:
  ⚖️ 법률 서비스 (권리분석, 계약검토, 명도, 배당)
  🧮 세무/회계 (세금계산, 절세, 검증, 신고)
  🏠 부동산 중개 (실사, 명도, 처분, 임대)
  📋 컨설팅 (투자전략, 포트폴리오, 시장분석)
  🏗️ 시행/시공 (개발, 리모델링, 시공)

서비스 카드:
  - 전문가 이름/법인명
  - 인증 배지 (자격증, 실적)
  - 전문분야 태그
  - 평점 (★ 4.8 / 23건)
  - 가격 범위
  - "견적 요청" 버튼

연결 방식:
  거래 프로세스 각 단계에서 자동 추천:
    NDA 단계 → 법률 서비스 추천
    실사 단계 → 중개사/컨설턴트 추천
    계약 단계 → 법무/세무 추천
    매입 후 → 시행사/시공사 추천
```

### 4.2 견적/의뢰 시스템
```
플로우:
  1. 투자자/기관이 "서비스 요청" 클릭
  2. 요청서 작성 (물건정보, 요구사항, 예산, 일정)
  3. 해당 분야 전문가에게 자동 알림
  4. 전문가가 견적서 제출 (가격, 일정, 범위)
  5. 의뢰자가 견적 비교 → 선택
  6. 서비스 진행 → 완료 → 평가

DB:
  service_requests (id, requester_id, category, listing_id, description, budget, deadline)
  service_quotes (id, request_id, provider_id, price, duration, scope, status)
  service_reviews (id, quote_id, rating, comment, created_at)
```

## 5. 수익 분배 생태계

### 5.1 모든 참여자가 수익을 얻는 구조
```
┌──────────────────────────────────────────────────────┐
│                    하나의 NPL 거래                     │
│                    (10억원 매각)                       │
├──────────────────────────────────────────────────────┤
│                                                      │
│  💰 거래 수수료 (1%)                                  │
│    └─ 매도자 0.5% (500만원) + 매수자 0.5% (500만원)    │
│                                                      │
│  🤝 영업 파트너 (0.1~0.3%)                            │
│    └─ 매물 소개: 100~300만원                           │
│    └─ 투자자 소개: 건당 50~100만원                     │
│                                                      │
│  ⚖️ 전문 서비스                                       │
│    └─ 법률 자문: 100~200만원                           │
```

### 5.2 참여자별 수익 모델
```

금융기관: 매물 등록 → 더 많은 입찰 → 최적가 매각 → 손실 최소화
투자자: 검색/분석 → AI 매칭 → 최적 매입 → 수익 극대화
법무법인: 서비스 등록 → 고객 유입 → 건별 수수료
세무법인: 서비스 등록 → 거래 연동 → 자동 의뢰
중개사: 지역 등록 → 실사 의뢰 → 건별 수수료
컨설턴트: 리서치 발행 → 구독자 확보 → 상담 수익
영업파트너: 리드 소개 → 거래 성사 → 소개 수수료
언론사: 데이터 활용 → 기사 → 트래픽 → 광고 수익
시행사: 물건 매칭 → 투자자 모집 → 개발 수익
```

## 6. 신뢰/평판 시스템

### 6.1 참여자 평판 점수
```
모든 참여자에게 플랫폼 내 평판 점수 부여:

점수 요소:
  - 거래 완료 건수 (누적)
  - 상대방 평가 (5점 만점)
  - 응답 속도 (평균)
  - 분쟁 이력 (감점)
  - 서비스 이행률 (전문가)
  - 활동 빈도 (월간)

등급:
  ⭐⭐⭐⭐⭐ 플래티넘 (상위 5%)
  ⭐⭐⭐⭐ 골드 (상위 20%)
  ⭐⭐⭐ 실버 (상위 50%)
  ⭐⭐ 브론즈 (나머지)
  ⭐ 신규 (3건 미만)

혜택:
  플래티넘: 수수료 할인 20%, 추천매물 우선 노출, 전용 딜룸
  골드: 수수료 할인 10%, 배지 표시
  실버: 기본 혜택
  신규: 온보딩 서포트, 첫 거래 수수료 면제

페이지:
  /reputation/my-score — 내 평판 점수
  /reputation/[user_id] — 상대방 평판 조회
```

### 6.2 상호 평가 시스템
```
거래 완료 후:
  매도자 → 매수자 평가 (거래 태도, 입금 속도, 소통)
  매수자 → 매도자 평가 (정보 정확도, 응답 속도, 투명성)
  의뢰자 → 전문가 평가 (전문성, 가격, 일정 준수)
  전문가 → 의뢰자 평가 (소통, 결제, 협조)

비공개 메모:
  평가자만 볼 수 있는 내부 메모 (다음 거래 참고)
```
