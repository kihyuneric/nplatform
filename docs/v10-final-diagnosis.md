## 4부: 사용자(대표님)가 직접 해야 할 것

### 즉시 (Phase A 전)
1. **Supabase 프로젝트 생성** → URL + anon key를 `.env.local`에 입력
2. **도메인 확보** → nplatform.co.kr (또는 원하는 도메인)
3. **Vercel 계정 생성** → GitHub 연동, 프로젝트 배포
4. **Anthropic API 키 발급** → Claude API 사용

### Phase C 시
5. **토스페이먼츠 가맹점 등록** → 사업자등록증 필요
6. **사업자 통장 개설** → 정산 수령용

### Phase D 시
7. **영어/일본어 번역 검수** → AI 번역을 네이티브가 검수

### 앱 출시 시
8. **Apple Developer 계정** ($99/년)
9. **Google Play Console** ($25 일회성)

### 기존 부동산 데이터 서비스 연동 시
10. **API 문서 공유** → 실거래가/시세/뉴스 API 스펙
11. **API 키 발급** → NPLatform 서버에서 접근 허용
