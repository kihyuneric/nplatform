## 1. 시장 기회 (Market Opportunity)

### 한국 NPL 시장 규모
| 지표 | 수치 | 출처 |
|------|------|------|
| 국내 부실채권 잔액 | **약 30조원** | 금융감독원 2025 |
| 연간 NPL 매각 거래액 | **약 8~12조원** | 업계 추정 |
| NPL 전문 투자자 수 | **약 5,000~10,000명** | 추정 |
| NPL 매각 금융기관 | **약 200+개** | 은행+저축은행+대부업 |
| 온라인화 비율 | **5% 미만** | 대부분 오프라인/이메일 |

### 시장 페인포인트 (Pain Points)
```
매도자 (금융기관):
  ❌ 매각 공고를 개별 투자자에게 이메일/팩스로 발송
  ❌ 입찰 접수를 엑셀로 관리
  ❌ 투자자 풀이 제한적 (기존 거래처만)
  ❌ 가격 발견(price discovery)이 비효율적

매수자 (투자자):
  ❌ 매물 정보를 구하기 어려움 (인맥 의존)
  ❌ 분석을 수동 엑셀로 진행 (2~3일 소요)
  ❌ 낙찰가율, 유사사례 데이터 없음
  ❌ 여러 금융기관 매물을 비교할 수 없음

시장 전체:
  ❌ 거래 투명성 부족
  ❌ 정보 비대칭 (기관 >> 개인)
  ❌ 거래 프로세스 표준화 안 됨
  ❌ 시장 데이터/통계 부재
```

### 해결 (NPLatform)
```
✅ 모든 금융기관의 NPL을 한 곳에서 검색
✅ AI가 2분 만에 분석 완료 (수동 2~3일 → 2분)
✅ 온라인 입찰/경매 → 가격 발견 효율화
✅ 매도-매수 AI 자동 매칭
✅ 거래 전 과정 디지털화 (NDA~계약~정산)
✅ 시장 데이터/인텔리전스 제공
```

## 2. VC가 투자하는 이유

### 2.1 TAM/SAM/SOM
```
TAM (전체 시장): 30조원 × 1% 수수료 = 3,000억원/년
SAM (접근 가능 시장): 12조원 × 1% = 1,200억원/년
SOM (초기 목표 시장): 1,200억 × 5% = 60억원/년 (3년차)
```

### 2.2 비즈니스 모델 (Revenue Model)
```
┌─────────────────────────────────────────────┐
│  수익 모델 (4가지)                            │
├──────────────┬──────────────────────────────┤
│ 거래 수수료   │ 매각 완료 시 매도/매수 각 0.5% │
│              │ 10억 거래 → 1,000만원 수수료   │
│              │ → 연 100건 × 평균 10억 = 10억  │
├──────────────┼──────────────────────────────┤
│ SaaS 구독    │ Pro: 월 50만원 / Enterprise:   │
│              │ 월 200만원                      │
│              │ → 100 기관 × 100만원 = 12억/년 │
├──────────────┼──────────────────────────────┤
│ 프리미엄 노출 │ 추천매물 배치: 건당 10~50만원   │
│              │ → 월 200건 × 20만원 = 4.8억/년 │
├──────────────┼──────────────────────────────┤
│ 데이터 API   │ 시장 데이터 API: 월 100~500만원 │
│              │ → 50 기관 × 200만원 = 12억/년  │
└──────────────┴──────────────────────────────┘

3년차 목표 ARR: 약 40억원
```

### 2.3 경쟁 해자 (Moat)
```
1. 네트워크 효과:
   매도자↑ → 매물↑ → 매수자↑ → 거래↑ → 매도자↑ (양면 플라이휠)

2. 데이터 해자:
   거래가 쌓일수록 AI 매칭/분석 정확도↑ → 경쟁사 추월 불가

3. 전환 비용:
   기관이 매물 등록/정산/히스토리를 NPLatform에 쌓으면 이탈 어려움

4. 브랜드/신뢰:
   NPL 거래 = 고액 + 전문성 → 신뢰 구축한 플랫폼이 독점

5. 기술 우위:
   AI OCR, 라이브 경매, 매칭 엔진 → 카피캣이 따라오기 어려움
```

## 4. 파괴력 모듈 (+165점)

### 🔥 D1. 실시간 거래소 모드 (Deal Exchange) — 30점
```
컨셉: 주식 거래소처럼 NPL을 실시간 거래

기능:
  - 실시간 호가창 (매도호가/매수호가 표시)
  - 체결 히스토리 (시간, 가격, 수량)
  - 시장가/지정가 주문
  - 차트: 캔들스틱 (일별 거래가 추이)
  - 거래량 표시
  - 알림: 내 지정가 체결 시 즉시 알림

왜 파괴적:
  현재 NPL 거래 = 수 주 소요 → 실시간 거래소 = 수 분 내 체결
  소액 분할 투자 가능 (10억 채권 → 1억 단위 분할)

페이지:
  /exchange — 거래소 메인 (호가창 + 차트 + 체결)
  /exchange/[listing_id] — 개별 매물 거래
  /exchange/portfolio — 내 보유 포지션
```

### 🔥 D2. NPL 분할 투자 (Fractional Investment) — 25점
```
컨셉: 고액 NPL을 소액으로 분할하여 공동 투자

기능:
  - 매도자가 "분할 투자 허용" 설정
  - 최소 투자 금액 설정 (예: 1,000만원부터)
  - 진행률 바 (모집 현황: 7.2억/10억, 72%)

  - 수익 분배 자동 (지분율 기준)
  - 공동 투자 계약 자동 생성

왜 파괴적:
  NPL 투자 진입장벽 = 수억~수십억 → 1,000만원부터 참여
  개인 투자자 시장 10배 확대

페이지:
  /invest/fractional — 분할 투자 매물 목록
  /invest/fractional/[id] — 투자 참여 (모집현황, 계약, 결제)
  /invest/my-positions — 내 투자 현황
```

페이지:
