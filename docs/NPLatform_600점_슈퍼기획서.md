### 🏦 S1. 매도자 포털 (Seller Portal) — 50점

> 금융기관, 대부업체, 자산관리회사가 NPL을 자유롭게 등록·마케팅·매각

#### S1-1. 매도자 온보딩 (10점)
- [ ] **기관 유형 선택**: 은행, 저축은행, 대부업체, 자산관리회사(AMC), 보험사, 증권사, 캐피탈, 기타
- [ ] **사업자 인증**: 사업자등록번호 실시간 검증 API (국세청 연동 또는 mock)
- [ ] **KYC 워크플로우**: 서류 업로드 → 관리자 심사 → 승인/반려 → 이메일 알림
- [ ] **기관 프로필 페이지**: 로고, 소개, 전문분야, 거래실적, 평점
- [ ] **권한 등급**: 일반(매물등록), 프리미엄(추천매물+배너), 엔터프라이즈(전용딜룸+API)
- [ ] 페이지: `/seller/onboarding`, `/seller/profile`

#### S1-2. 매물 등록 고도화 (15점)
- [ ] **단건 등록 위자드 (6단계)**:
  1. 기본정보 (채권유형, 담보유형, 주소)
  2. 채권정보 (원금, 이자, 연체기간, 설정금액)
  3. 담보정보 (면적, 용도, 건축년도, 공시가격)
  4. 경매/법원 정보 (사건번호, 법원, 매각기일)
  5. 매각조건 (희망가, 입찰방식, 공개수준, NDA 필요여부)
  6. 마케팅 설정 (사진, 설명, 추천태그, 노출기간)
- [ ] **대량 등록**: 엑셀 업로드 (기존) + CSV + API 연동
- [ ] **임시저장/자동저장**: 5분마다 자동저장, 초안 관리
- [ ] **매물 복제**: 기존 매물 기반 신규 등록
- [ ] **포트폴리오 번들 등록**: 여러 매물을 하나의 포트폴리오로 묶어서 일괄 매각
- [ ] 페이지: `/seller/listings/new`, `/seller/listings/bulk`, `/seller/portfolio/new`

#### S1-3. 매도자 대시보드 (15점)
- [ ] **내 매물 관리**: 상태별 필터 (초안/활성/입찰중/거래중/매각완료)
- [ ] **매물별 성과**: 조회수, 관심등록수, 문의수, 입찰수 추이 차트
- [ ] **입찰 관리**: 받은 입찰 목록, 입찰자 정보, 협상 도구
- [ ] **수수료 정산**: 거래완료 매물의 플랫폼 수수료 내역, 세금계산서
- [ ] **마케팅 도구**: 매물 홍보 배너 만들기, SNS 공유 링크 생성
- [ ] **알림 센터**: 새 입찰, 문의, 입찰마감, 거래완료 알림
- [ ] 페이지: `/seller/dashboard`, `/seller/analytics`, `/seller/settlement`

#### S1-4. 마케팅/노출 관리 (10점)
- [ ] **추천 매물 신청**: 유료 프리미엄 노출 (홈페이지 상단, 검색 상단)
- [ ] **태그 기반 노출**: 자동 태그 (급매, 신규, 인기, 협상가능) + 커스텀 태그
- [ ] **매물 상세 SEO**: 자동 meta description, 구조화 데이터 (JSON-LD)
- [ ] **매물 공유**: 카카오톡 공유, URL 단축, QR코드 생성
- [ ] **이메일 마케팅**: 관심 투자자에게 신규 매물 이메일 발송
- [ ] **실적 리포트**: 월간 매도 실적 PDF 보고서

### 💰 S2. 매수자 포털 (Buyer Portal) — 50점

> 대부업체, 자산운용사, 개인투자자가 원하는 NPL을 찾고 거래

#### S2-1. 매수자 온보딩 (10점)
- [ ] **투자자 유형 선택**: 대부업체, 자산운용사, 개인투자자, 법인투자자, 공동투자
- [ ] **투자 프로필**: 관심 담보유형, 관심 지역, 투자 예산 범위, 투자 경험
- [ ] **인증 등급**: 일반(검색/분석), 인증(입찰참여), 프리미엄(딜룸+AI)
- [ ] **투자성향 진단**: 5문항 설문으로 보수적/중립/공격적 분류
- [ ] 페이지: `/buyer/onboarding`, `/buyer/profile`

#### S2-2. 맞춤 매물 추천 (15점)
- [ ] **AI 매칭 엔진**: 투자 프로필 기반 매물 자동 추천
  - 담보유형 매칭 (30%), 지역 매칭 (25%), 금액대 매칭 (20%), 할인율 매칭 (15%), 회피조건 (10%)
- [ ] **추천 피드**: 매일 갱신되는 "오늘의 추천" 매물 3~5건
- [ ] **알림 설정**: 조건 매칭 매물 등록 시 즉시 알림 (이메일/푸시/카톡)
- [ ] **수요 설문 고도화**: 상세 투자 조건 입력 → AI 매칭 → 결과 리포트
- [ ] **관심매물 워치리스트**: 가격/상태 변동 알림, 메모 기능
- [ ] 페이지: `/buyer/recommendations`, `/buyer/watchlist`, `/buyer/alerts`

#### S2-3. 투자 분석 도구 (15점)
- [ ] **포트폴리오 분석**: 보유/관심 NPL 전체 수익률, 리스크 분석
- [ ] **비교 분석 강화**: 최대 6건 병렬 비교 (레이더 차트, 테이블)
- [ ] **실사 체크리스트**: AI 생성 실사 항목 (법률, 담보, 재무, 점유)
- [ ] **투자 시나리오**: 보유기간/대출비율/매도가별 수익 시뮬레이션
- [ ] **시장 인사이트**: 지역별/유형별 트렌드, 낙찰가율 추이, 경쟁도
- [ ] 페이지: `/buyer/portfolio`, `/buyer/compare`, `/buyer/due-diligence`

#### S2-4. 거래 도구 (10점)
- [ ] **1:1 문의**: 매도자에게 직접 메시지 (비공개 딜룸 자동 생성)
- [ ] **가격 제안**: 매도자에게 가격 제안서 전송
- [ ] **LOI (의향서)**: 구매 의향서 온라인 작성 및 전송
- [ ] **거래 진행 추적**: 내가 참여중인 모든 거래 타임라인
- [ ] 페이지: `/buyer/inquiries`, `/buyer/offers`, `/buyer/deals`

### 🤝 S3. 거래 매칭/중개 플랫폼 (Marketplace Engine) — 40점

> 매도자와 매수자를 자동으로 연결하는 양면 플랫폼

#### S3-1. 양면 매칭 (15점)
- [ ] **매도 등록 → 매수 매칭**: 매물 등록 시 관심 투자자 자동 알림
- [ ] **매수 수요 → 매도 매칭**: 수요 설문 등록 시 보유 매물 자동 추천
- [ ] **매칭 대시보드**: 매칭 점수, 관심도, 컨택 상태 시각화
- [ ] **양방향 관심 표시**: 매도자도 매수자의 관심을 확인 가능
- [ ] **매칭 히스토리**: 지난 매칭 결과, 성사율 통계

#### S3-2. 입찰 마켓플레이스 (15점)
- [ ] **공개 입찰**: 경쟁 입찰 (최고가 낙찰)
- [ ] **비공개 입찰**: 초청 입찰 (매도자가 선택한 매수자만 참여)
- [ ] **수의 계약**: 1:1 협상 (가격 제안/역제안)
- [ ] **포트폴리오 입찰**: 복수 매물 일괄 입찰
- [ ] **입찰 이력/통계**: 기관별 입찰 성사율, 평균 낙찰가율
- [ ] **입찰 캘린더**: 마감 임박 순 캘린더 뷰

#### S3-3. 거래 프로세스 관리 (10점)
- [ ] **7단계 거래 흐름**: 관심표시 → NDA → 실사 → 가격협상 → LOI → 계약 → 잔금/이전
- [ ] **각 단계별 체크리스트**: 필요 서류, 승인 항목, 기한
- [ ] **에스크로 연동 (mock)**: 거래 대금 안전 결제 UI
- [ ] **거래 완료 리포트**: 거래 요약, 수수료, 세금 안내

## 핵심 사용자 여정 (User Journey)

### 📍 매도자 여정
```
1. 회원가입 → 기관유형 선택 → KYC 인증
2. 매물 등록 (단건/대량/포트폴리오)
3. 마케팅 설정 (태그, 노출, 추천)
4. 입찰 수신 → 입찰자 검토 → 낙찰 확정
5. 딜룸에서 협상 → NDA → 실사 → 계약
6. 거래 완료 → 정산 → 실적 리포트
```

### 📍 매수자 여정
```
1. 회원가입 → 투자유형 선택 → 투자성향 진단
2. AI 추천 매물 확인 / 직접 검색
3. 관심매물 저장 → AI 분석 → 수익률 시뮬레이션
4. NDA 서명 → 상세 열람 → 실사
5. 가격 제안 / 입찰 참여
6. 딜룸 협상 → LOI → 계약
7. 거래 완료 → 포트폴리오 관리
```
