## 0. Executive Summary

NPLatform은 국내 유일의 **AI 기반 NPL(부실채권) 거래·분석 전문 플랫폼**이다. 2026 상반기 현재 283 페이지 · 242 API · 216 컴포넌트 · 16 DB 마이그레이션으로 구성된 Next.js 15 + Supabase 스택 위에 Claude API, pgvector RAG, OCR, 경매 시뮬레이터, 딜룸을 통합한 **end-to-end 거래 인프라**를 구축했다.

본 로드맵은 향후 12개월(2026 Q2 → 2027 Q2)에 걸쳐 다음 네 가지 목표를 달성하기 위한 통합 실행 계획이다.

| 목표 | 지표 | 현재 | 2027 Q2 |
|------|------|------|---------|
| GMV (총 거래액) | 누적 원화 | ₩2,847억 | ₩8,000억 |
| MAU | 월간 활성 사용자 | 1,900명 | 12,000명 |
| 수익 | 연환산 매출(ARR) | ₩18억 | ₩75억 |
| 글로벌 기술 점수 | 내부 평가 | 53.6/100 | 75/100 |

**핵심 전환 테제**
1. **역할 기반 2-stage 온보딩** — 매각사(금융기관/대부/AM/일반법인) · 투자그룹(일반/전문)로 가입 트래픽을 수수료·기능 티어와 정렬
2. **PNR(Priority Negotiation Right) 과금 분리** — NPL 1.5% + 우선협상 0.3%p / 부동산 0.9% / 매각사 6개월 무료로 단가 예측 가능성 확보
3. **AI Copilot + pgvector RAG** — 법률·판례·시세 DB를 Claude 위에 얹어 "분석 소요 시간 4시간 → 27초"를 마케팅 히어로로 사용
4. **글로벌 기술 점수 75점 돌파** — LightGBM 가격 예측, 실시간 입찰 WebSocket, Jest 80% 커버리지를 Phase 2~4 내 순차 확보

## 1. 비즈니스 전략

### 1.1 시장 정의와 TAM/SAM/SOM

| 지표 | 규모 (2026 추정) | 근거 |
|------|------------------|------|
| 국내 NPL 시장 연간 거래 규모 (TAM) | ₩12~15조 | 금감원 은행권 부실채권 잔액 + 저축은행/대부 이관분 |
| 디지털화 가능 세그먼트 (SAM) | ₩3~4조 | 건당 20억 이하 담보부 NPL 중심 (소형·중형) |
| 2027년 타겟 점유율 (SOM) | ₩8,000억 GMV (2.0~2.5%) | Phase 2 종료 시점 목표 |

**세그먼트 우선순위**
1. **1차 (Q2 2026)** — 저축은행·캐피탈의 소형 담보부 NPL (건당 2~10억). 기존 오프라인 네트워크 의존도 낮아 디지털 전환 유인 강함
2. **2차 (Q3 2026)** — 중형 개인 투자자 + 전문 AM 주니어 딜러. "AI 분석 + 경매 시뮬" UX 차별화로 락인
3. **3차 (2027)** — 시중은행 중형 포트폴리오 일괄 매각 (₩50~500억 규모). PNR + 딜룸 + 에스크로 풀패키지로 경쟁

### 1.2 포지셔닝

| 축 | 우리 | 기존 플랫폼 (옥션사이트) | 오프라인 브로커 |
|----|-----|------------------------|--------------|
| 데이터 깊이 | AI + RAG + OCR 통합 | 목록만 | 개인 네트워크 |
| 거래 완결성 | NDA→LOI→에스크로→전자계약 원스톱 | 연락처 교환 | 수작업 |
| 수수료 투명성 | 고정률 공개 (1.5% / 0.9%) | 불투명 | 2~5% 협상 |
| 접근 통제 | L0~L3 4단계 + PII 자동 마스킹 | 기본 회원제 | 없음 |

**한 문장 포지셔닝**
> "NPL 매각사와 투자자를 직접 연결하는, AI가 분석하고 법이 보호하는 유일한 거래소."

### 1.3 성장 엔진 (5+1 프레임)

1. **Supply-side moat** — 매각사 첫 6개월 수수료 무료 + 일괄 등록 OCR로 공급 트리거
2. **Demand-side moat** — 무료 L0 카드 탐색 + AI 분석 리포트 월 3건 무료로 수요 트리거
3. **Network effect** — 입찰 참여 투자자 수가 매각사 평균 낙찰가를 올리는 정비례 구조
4. **Data flywheel** — 낙찰 데이터 → 가격 예측 모델 재학습 → 분석 정확도 ↑ → 신규 유입 ↑
5. **Regulatory moat** — 금감원 핀테크 혁신 인증, NDA 전자서명, AES-256, ISO 27001로 B2B 진입장벽 상승
6. **AI moat** — Claude + 자체 NPL 코퍼스(28,391건 분석 로그) 기반 Copilot이 경쟁사 복제 지연 요소

### 2.4 Growth Loops

**Loop A — 매각사 공급 확대**
매각사 가입 → 무료 6개월 → 3건 이상 등록 → 낙찰 체결 → 레퍼런스 케이스 스터디 → 영업자료 → 유사 매각사 영업

**Loop B — 투자자 바이럴**
AI 분석 리포트 공유 기능 (PDF/링크) → 수신자 클릭 → 가입 → 본인 분석 → 재공유

**Loop C — 파트너 레퍼럴**
변호사·세무사 파트너 등록 → 자문 딜 연결 → 딜 성공 시 20% 수수료 공유 → 전문가 네트워크 → 전문가가 고객사에 플랫폼 추천

## 3. 수익 모델

### 3.1 현행 수수료 구조 (2026 Q2 적용)

| 거래 유형 | 매수자 기본 | PNR 추가 | 투자그룹 할인 | 매각사 |
|----------|-----------|---------|-------------|-------|
| NPL (부실채권) | 1.5% | +0.3%p | 일반 -0.05%p / 전문 -0.1%p | 첫 6개월 무료, 이후 0.5% |
| 부동산 담보 직접 거래 | 0.9% | +0.3%p | 동일 | 동일 |

**실효 수수료 예시 (NPL 10억 딜, 전문 투자그룹 + PNR)**
- 매수자: 10억 × (1.5% − 0.1%p + 0.3%p) = 10억 × 1.7% = **₩1,700만**
- 매각사 (6개월 후): 10억 × 0.5% = **₩500만**
- 플랫폼 총수익: **₩2,200만 / 건**

### 3.2 부가 수익원 (5개 스트림)

| 스트림 | 모델 | 2026 Q4 목표 |
|-------|------|------------|
| ① 거래 수수료 | 건당 % | ARR 45억 (월 ₩3.75억) |
| ② 프리미엄 구독 (전문 투자그룹) | ₩49만/월 · 개인, ₩180만/월 · 법인 | 구독자 300 → ARR 10억 |
| ③ AI 분석 건별 과금 (초과분) | 월 무료 3건 초과 시 ₩29,000/건 | ARR 5억 |
| ④ B2B API (시세·분석 API) | ₩500만/월 기본 + 콜당 과금 | 파트너 15개사 → ARR 12억 |
| ⑤ 파트너 자문 수수료 (레퍼럴 공유) | 딜당 파트너 → 플랫폼 5% 레퍼럴 수익 | ARR 3억 |

### 3.3 단위 경제 (Unit Economics)

**투자자 코호트 (개인 일반 가정)**
- CAC (획득 비용): ₩80,000 (SEM + 콘텐츠 혼합)
- ARPU 월: ₩32,000 (수수료 평균)
- Gross margin: 78%
- Payback: 3.2개월
- LTV (24개월 유지): ₩610,000
- **LTV/CAC = 7.6** ✅ 건강

**매각사 코호트**
- CAC: ₩1,200,000 (B2B 세일즈)
- 6개월 무료 후 평균 매각 건수: 4건/분기
- 평균 거래액 8억 × 0.5% = ₩400만/건
- 분기 ARPU ₩1,600만
- Payback: <1분기
- **CAC 회수 기간 극단적으로 짧음 — 공격적 영업 정당화**

### 3.4 Pricing 실험 파이프라인

- 입장 애니메이션: stagger 0.08s, y:24→0 + opacity 0→1
- 프리퍼스-reduced-motion 자동 존중

## 부록 B. 용어 사전

- **L0~L3**: 접근 단계. L0 기본 공개 / L1 본인인증 / L2 NDA / L3 LOI 승인
