## 글로벌 유니콘 로드맵

### Phase A — 지능형 분석 엔진 고도화 (2026 Q2-Q3)
> **목표**: 데이터만 넣으면 즉시 투자 결정 가능한 수준의 AI 분석

#### A-1. 실시간 시장 데이터 통합
```
법원경매 오픈API ──→ 자동 매물 수집 (1일 1회 배치)
국토부 실거래가 ──→ 시세 자동 업데이트 (Redis TTL 24h)
KB부동산 시세 ───→ 가격 추정 정밀도 향상 (+15%)
```

**구현 파일:**
- `lib/market-data/auction-court.ts` — 법원 API 연동
- `lib/market-data/molit-api.ts` — 국토부 실거래가
- `app/api/v1/market-data/sync/route.ts` — 수동/자동 동기화

#### A-2. ML 모델 정밀화
```
현재: Linear Regression (정확도 78%)
목표: Gradient Boosting XGBoost (목표 88%)
데이터: 5,000건 이상 → 정확도 자동 향상 (Active Learning)
```

**피처 엔지니어링:**
- 위경도 → 학군/역세권/편의시설 자동 계산
- 시계열 특성 (계절성, 금리 사이클)
- 텍스트 특성 (등기부등본 키워드 분석)

#### A-3. 자동 분석 파이프라인
```
사용자 데이터 입력
    ↓
OCR (등기부등본 자동 파싱) ← 이미 구현
    ↓
자동 분석 엔진 실행 (6개 모델)  ← 이번 구현
    ↓
AI 종합 리포트 생성 (Claude)
    ↓
투자 의사결정 추천 + 근거 제공
```

**KPI**: 분석 완료까지 < 30초 (현재 수동 2-3시간)

### Phase B — 거래 자동화 플랫폼 (2026 Q3-Q4)
> **목표**: NPL 거래 전 과정의 디지털화 — 계약서까지 AI가 생성

#### B-1. 스마트 매칭 알고리즘 v3
```
투자자 프로파일 (선호 유형/지역/수익률/리스크)
    ↓
임베딩 기반 유사도 매칭 (16-dim → 64-dim)
    ↓
Real-time 추천 피드 (WebSocket)
```

**구현:**
- `lib/ml/matching-engine-v3.ts`
- `app/api/v1/matching/stream/route.ts` — SSE 스트리밍

#### B-2. AI 계약서 자동 생성
```
거래 조건 입력 (채권양수도, 금액, 담보 정보)
    ↓
Claude AI → 표준 계약서 초안 생성
    ↓
법무사 검토 워크플로우 (선택)
    ↓
전자서명 (토스인증서 연동)
    ↓
계약 완료 자동 기록
```

#### B-3. 디지털 딜룸 고도화
현재 딜룸에 추가:
- 체크리스트 자동 생성 (물건 유형별)
- 문서 버전 관리 (타임스탬프)
- 다자간 협상 로그 (Operational Transformation)
- 법무법인 API 연동 (선택적 법률 검토)

### Phase C — 데이터 네트워크 효과 (2027 Q1-Q2)
> **목표**: 경쟁사가 복제 불가능한 독점 데이터 자산 구축

#### C-1. NPL 가격 지수 (NPLatform Price Index)
```
플랫폼 내 모든 거래 데이터 집계
    ↓
지역별·유형별 NPL 할인율 지수 발표 (월간)
    ↓
언론·기관투자자 레퍼런스 데이터로 포지셔닝
    ↓
유료 데이터 API 제공 → 새로운 수익원
```

**구현:**
- `app/api/v1/indices/npl-price/route.ts`
- 공개 대시보드: `/market/indices`

#### C-2. 예측 인텔리전스
```
거래 완료 데이터 누적
    ↓
모델 자동 재학습 (Active Learning)
    ↓
정확도 자동 향상: 78% → 88% → 93% (데이터 증가에 따라)
```

**임계점**: 10,000건 거래 완료 시 정확도 90% 돌파 예상

#### C-3. API 마켓플레이스
```
외부 개발자 → NPLatform API 구독
    ↓
분석 결과, 시장 데이터, 추천 API 판매
    ↓
은행·증권사·PE펀드 B2B 계약
```

## 핵심 차별화 포인트

### 1. Auto-Analysis Engine (최우선)
> **"데이터를 넣으면 즉시 분석 완료"**

```
경쟁사: 분석가가 수동으로 2-3시간
NPLatform: AI가 30초 내 6개 모델 동시 분석
```

구현 완료:
- `lib/analysis-engine.ts` — 6개 모델 자동 실행
- `app/api/v1/analysis-engine` — REST API
- `app/(main)/admin/ml` — 관리자 데이터 요구사항 패널

### 2. Real-time Deal Flow
```
매물 등록 → AI 자동 분석 → 투자자 매칭 → 딜룸 자동 생성 → 계약 체결
(수동 개입 최소화)
```

### 3. Proprietary Data Moat
```
거래 누적 → 모델 정확도 향상 → 더 좋은 분석 → 더 많은 거래 → 반복
(데이터 네트워크 효과)
```
